- Doom 3: BFG Edition (1.1400.346456) - [[Steam](http://store.steampowered.com/app/208200/)];
- RBDOOM-3-BFG (1.1.0-preview3-7728dc3) - [[github](https://github.com/RobertBeckebans/RBDOOM-3-BFG/releases/tag/1.1.0-preview3)].
